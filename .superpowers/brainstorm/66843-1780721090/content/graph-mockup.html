How this works

  • Each metric gets a compact sparkline (28px tall)
  • Live value displayed to the right of the label
  • 1-minute window of streaming data with gradient fill
  • Recharts AreaChart for implementation (already in deps)
  • Zero-axis reference line on roll/pitch for visual clarity

What's NOT graphed

  • Navigation (lat/lon/alt/gs/vs/hdg/sats/hdop) — text remains
  • Power (bat%/V/A) — text remains, could add battery trend later
  • Link status (TEL/PKT/VID dots) — text remains
