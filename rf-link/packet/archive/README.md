# Archive — Historical Reference Only

These files document the project's evolution from hello-world BPSK through
the enhanced codec development. They are not in active use but are preserved
for reference.

## Contents

- `grc-blocks/` — Original GNU Radio-generated hierarchical blocks
- `hello-world/` — Original (unprotected) BPSK hello chain
- `experimental-rx/` — Offline decode iterations that led to pkt_enhanced_rx.py
- `gr-rx/` — GRC-based / native GNU Radio RX scripts  
- `gr-tx/` — GRC-based / minimal TX scripts
- `experiments/` — One-off test scripts

See ../README.md for the active project structure.
